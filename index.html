本站叫「狸貓的在線辭典 2.0 版」(Leimaau's Webdict 2.0)，上線於2020年02月，是一個私人在線辭典，旨在記錄自己的學習筆記。相對於 1.0 版的 Leimaau's Webdict 本版有了較大的改進，程序的開發和製作得益於 sgalal 和匆匆的我的技術支持，特此致謝。

本站原先開發和運行於 github 中，爲了避免網絡不穩定，故將程序備份於 netlify 中運行，因此本站有兩個地址，github 地址作主䈎地址，netlify 地址作備份地址。關於生僻字的顯示，可在字體試驗䈎查看。「標準音經驗回歸值」爲審音表標※的人工經驗讀音，「標準音統計回歸值」爲統計佔比大於等於《廣韻》異讀數所佔百分比的讀音，無異讀或兩個異讀設置爲不低於40%的讀音。

字典中計算資料權重的幾何平均值時對各資料權重的預設值爲：
9(2018)；
7(1998、2008、2009)；
5(1997、2003、2021)；
3(1994、2000、2007、201703、201705)；
1.5(201806、2022)
